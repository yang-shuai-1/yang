## 产品背景

远程协作工具很多，但大部分白板工具的绘画体验和协作延迟都不理想。FlowBoard 的核心理念是"让画笔感觉像在纸上一样"。

## 技术亮点

- 基于 Canvas API 自研渲染引擎，支持 60fps 流畅绘画
- WebSocket 实时同步，多人协作延迟 < 50ms
- 手绘图形识别为规范几何形状
- 支持无限画布和视口管理

## 设计系统

UI 极简克制，所有工具栏在需要时才出现，最大化画布空间。色彩方案采用暖灰底 + 单色强调，与整体品牌一致。
